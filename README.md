Student Grade Management System

📌 Overview
This project is a console-based Java application developed as Task 1 of my CodeAlpha Java Programming Internship.

The application accepts student names and scores, stores them using an ArrayList, and generates a report by calculating the average, highest, and lowest scores.

🚀 Features
- Accepts the number of students.
- Takes student names and scores as input.
- Stores student records using ArrayList.
- Calculates the average score.
- Finds the highest score.
- Finds the lowest score.
- Displays a complete student grade report.

 🛠 Technologies Used
- Java
- Core Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

📚 Concepts Covered
- Classes and Objects
- Constructors
- Collections Framework (ArrayList)
- Loops
- Conditional Statements
- User Input Handling
- Data Processing

 ▶️ How to Run

1. Compile the program:
   ```bash
   javac StudentGradeManagement.java
   ```

2. Run the program:
   ```bash
   java StudentGradeManagement
   ```
